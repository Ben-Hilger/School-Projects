Description of Website

  • This is a client front-end interface providing an assortment of options for the user to explore.
  • This interface provides the following:
    • Calculator
      • Allows you to perform + - * / operations on two numbers of your choosing
    • 7-day Weather Forcast
      • Provides up-to-date weather forcasting for the next 7 days based on a specified zipcode
    • Updating Sensors
      • Allows you to add a new sensor reading to the database for storage
    • Sensor Reports
      • Allows you to generate a sensor report with a specified number of sensor readings and optional location and sensor specification
